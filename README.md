## Deployment 

To deploy this project run

Vercions
- python3 -> 3.12
- node -> v20.11.1
- npm -> 10.2.4

Clone https://github.com/drosell271/NaturTeam-HackForGood2024

Backend
```bash
  $ python3 -m venv venv
  $ source venv/bin/activate
  $ pip install -r requirements.txt
  $ cd Backend
  $ python app/main.py
``` 
Frontend
```bash
  $ npm i
  $ npm start
```
